Javascript and DOM Browser App

I chose to create a random winner generator as my browser app utilizing Javascript and DOM (and HTML) as shown in the image above.

In script.js, we start by declaring variables that we want to manipulate using Javascript. We are able to do this by using the id relation established in index.html. For example, we are able to manipulate the 'Select Winner' button in the Javascript file by assigning its variable as:

var button = document.getElementById("button");

Below is the list of variables established above script.js. It also includes arrays that we will utilize to select a random name and background.

addEventListener

When we click on 'Submit Name', we want to add these names to be included in the list that the winner is retrieved from. When we click on 'Select Winner' we want a winner to be chosen from that list. Lastly, when we click on the background, we want it to change. All of this is made possible by adding an event listener to each one like the example below:

banners.addEventListener("click", changeBg);

Now, when the banner, or background of the winner announcement is clicked, it will invoke the changeBg function as shown below.

addNames

This function is invoked by addName.addEventListener("click", addNames);.

If the input value is more than 1, in other words, is not empty, we take the input and push it into names, the array that we've declared early in the script. After, the line, document.querySelector('form').reset(); resets the form so additional names can be added. We then include this list into local storage using set item.

Select Winner

Last, we adjust our script file to show us the winner from the given list. Here, we attach addEventListener to our button to do so upon being clicked. This is done similarly to the previous two examples usng Event Listener, but the function is directly passed in it. If the user does not submit any names, nothing changed and the user is notified. Otherwise, the names are listed and the before variable (that holds the placeholder of "Winner is...") is changed to an empty string to make room from the result. The winner is chosen randomly using (the same way we chose random backgrounds): var rando = names[Math.floor(Math.random()*names.length)]; Answer's innerText is now rando, the variable that will select random name from the names array.
