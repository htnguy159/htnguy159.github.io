Instagram Clone iOS

Instagram-like app that allows users to take and post pictures. Supports user staying logged in across restarts, logging out, viewing and adding new comments. Utilizes a custom backend database using Parse to store and retrieve user posts.

Tip Calculator iOS

An iOS app to make tipping easy. User can enter a bill amount, choose a tip percentage, and see the tip and total values. User can select between tip percentages by tapping different values on the segmented control and the tip value is updated accordingly.

Flixster iOS

An iOS app that allows users to browse movies now playing in theaters--similar to Fandango and Rotten Tomatoes. User can view the movie poster image and description for each movie.
